Release 2.3 - 23 Jun 2026
-------------------------
* Windows 11 compatibility: link discovery now works with "Core Isolation >
  Memory Integrity" (HVCI) enabled
* Capture now uses the in-box Windows Packet Monitor (pktmon) when available,
  instead of the legacy third-party capture driver that HVCI blocks
* The bundled tcpdump is now only used to decode captures offline (no driver)
* Added a clear warning when the legacy capture path is blocked by Core Isolation
* Falls back to the original live-capture method on Windows without pktmon
